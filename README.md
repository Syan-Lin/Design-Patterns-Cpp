# Design-Patterns-Cpp
C++11 实现设计模式

### 创建型模式
创建型模式抽象封装了实例化的过程，用户无需知道所使用对象具体是如何实例化的以及具体的对象细节，只需要知晓抽象类所定义的接口即可。而因为所使用的是接口，也就带来了松耦合，依赖接口而不是依赖过程。
